How should the community threads be structured?

Pick the layout that best fits the NYSC Diary community feel

A

Nairaland Style — Category Board

One sticky "General" mega-thread sits at the top. Below it, users can start topic threads under categories like Camp Life, PPA, Allowance, Jobs. Each thread has a title, email, and body. Replies nest under threads.

Pros

  • Familiar to Nigerians
  • Easy to scan topics
  • Great for discoverability

Cons

  • Needs category management
  • More complex to build
B

Single Feed + Pinned "Central Thread"

One pinned megathread (e.g. "NYSC General Chat") is always at the top. Below it, users post new threads freely — no categories. It's one scrollable feed. Threads show email (masked), title, and a reply count.

Pros

  • Simplest to build
  • Very easy to use
  • Everything in one place

Cons

  • Can get noisy
  • No topic filtering
C

Topic Tags + Feed (Reddit-lite)

Users post threads with an optional tag (Camp Life, Jobs, Allowance, Rant, etc.). The feed shows all threads but can be filtered by tag. Pinned central thread stays visible at top regardless of filter.

Pros

  • Flexible & organized
  • Good for growth
  • No hard categories needed

Cons

  • Slightly more complex UI
  • Tags need to be curated
